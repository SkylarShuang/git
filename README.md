# git 学习笔记
**git log** 查看提交历史

::HEAD::表示当前版本，::HEAD^::表示上一个版本，::HEAD^^::表示上上版本，::HEAD～100::表示上一百个版本

**git reflog**  查看命令历史

**git merge** 命令合并指定分支和当前分支
